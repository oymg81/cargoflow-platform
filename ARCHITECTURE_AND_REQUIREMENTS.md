This project has TWO objectives:

## 1. Client Website (Rosa / LOGISTI-K)
Build a premium modern logistics website for LOGISTI-K Freight Services Inc.

- preserve the client's original branding/colors
- modernize the company's online presence
- increase trust and professionalism
- support future logistics features

https://www.logisti-k.us/

---

## 2. Future SaaS Platform (CodingSoft)

The project must ALSO be architected as the foundation for a future reusable logistics SaaS platform.

This means:
- reusable backend architecture
- modular shipment systems
- scalable dashboard infrastructure
- future multi-company support

The public website is client-branded.
The internal platform/dashboard is CodingSoft-branded.

This separation is EXTREMELY IMPORTANT.
